# Tech Stack

## Frontend
- **Framework**: Next.js 15 with App Router
- **React**: Version 19
- **Language**: TypeScript with strict mode
- **Styling**: Tailwind CSS v4 with shadcn/ui components
- **UI Components**: Radix UI primitives (@radix-ui/react-*)
- **Icons**: Lucide React + React Icons
- **Drag & Drop**: @dnd-kit (core, sortable, utilities)
- **PWA**: next-pwa for Progressive Web App functionality

## Backend & Database
- **Authentication**: Supabase Auth with SSR support (@supabase/ssr)
- **Database**: Supabase PostgreSQL
- **API**: Next.js Server Actions + Supabase client
- **Edge Functions**: Supabase Edge Functions (task reset functionality)

## Development & Build Tools
- **Package Manager**: npm
- **Build Tool**: Next.js with Turbopack (dev mode)
- **Linting**: ESLint with Next.js config
- **TypeScript**: Version 5 with Next.js plugin
- **Deployment**: OpenNext for Cloudflare (@opennextjs/cloudflare)
- **Cloudflare**: Wrangler for CF operations

## Key Dependencies
- **Utilities**: clsx, tailwind-merge, class-variance-authority
- **Animation**: tw-animate-css
- **Date Handling**: Built-in JS with JST timezone conversion
